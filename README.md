# Yandex Academy Contest
Этот проект был сделан для участия в конкурсе Яндекс Академи (ШБР) учеником НИШ города Тараз Кубеевом Абылаем.
Проект использует технологию Gunicorn как замена сервера разработки. При этом Nginx не используется.

Для подключения проекта нужно проверить не используется ли нужный порт.
Для этого надо написать код

`sudo lsof -i tcp:8080`

Если порт не свободен то через PID процесса нужно его завершить.

`sudo kill -9 <PID>`

Затем если сервис не подключен надо включить проетк через следующую строчку кода:

`gunicorn --bind 0.0.0.0:8080 Contest.wsgi`

Где `0.0.0.0:8080` это заданный порт в заданиях.
